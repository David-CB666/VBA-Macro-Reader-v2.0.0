# Contributing

Thanks for your interest in contributing to VBA-Macro-Reader!

## Getting Started

1. Fork the repository.
2. Clone your fork:
   ```
   git clone https://github.com/YOUR_USERNAME/VBA-Macro-Reader-v2.0.0.git
   ```
3. Create a feature branch:
   ```
   git checkout -b feature/your-feature-name
   ```

## Development Setup

```bash
pip install -r requirements.txt
```

## Coding Guidelines

- Follow PEP 8 style for Python code.
- Add type hints where possible.
- Write clear, descriptive commit messages.
- Update documentation when modifying interfaces.

## Submitting Changes

1. Push your branch to your fork.
2. Open a Pull Request against the `main` branch.
3. Describe the changes and link any related issues.

## Reporting Issues

- Use the GitHub Issues tracker.
- Include steps to reproduce, expected behavior, and actual behavior.
- Provide your environment details (OS, Python version).
